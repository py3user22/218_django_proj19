__#0119 form submission
----------------------ex. Login form
__#note: The HTTP POST method is more secure than the HTTP GET method.
When a form is submitted using the Post method, the form data is inserted into the content of the HTTP request instead of appended at the end of the URL as is done with the GET method.

< form method="GET">
| .. < label for="username"> Username: < /label>
| .. < input type="text" id="username" name="username" value="admin">
| .. < label for="password"> Password: < /label>
| .. < input type="password" id="password" name="password" value="password">
| .. < input type="submit" value="Log In">
< /form>


___#0119 the form submission is determined by 2 attributes: action, method
--------------------------
action attribute
--> specifies to which web address the form is to be sent
--> can be full URL address, absolute path, or reletive path.


< form action="/login"> < /form>
