__#0121 javascript selectors .. to find specific objects
----------------------------
querySelector()
This querySelector() method returns the first element in the document that matches the selector

querySelectorAll()
This querySelectorAll() method returns all elements in the document that match the selector.

getElementById()
This getElementById() method returns a specific element whose id matches the selector.

getElementsByClassName()
This getElementsByClassName() method returns all elements in the document that has the class specified in the selector.
