__# global & local scope
-----------------
Scope
--------relates to code accessibility, determines which part of code are accessible by other parts
global scope is outside and above functions
local scope is inside the functions

__# block scope
-------------------
a variable declared in a block of code is only accessible inside that block.
all other code outside the block cant access it.
block scope is auto made when use let, const to make a variable


| Global scope |
let color = 'red';

| Block scope is inside the { } brackets
if (color == 'red') {
| .. let color == 'blue'
}


__# notes about let, const
tip: use let if the value might change in the future
use const if the value will never change
both let/const you cant use prior to declaration of.
